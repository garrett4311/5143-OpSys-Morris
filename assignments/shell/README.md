# Shell Project 02/18/2019
# Ben Shelton & Garrett Morris
The goal of this project was to design a shell that runs
many of the basic commands that are associated with a shell. 


### List of commands implemented

```
ls (-l, -h)
    not working: -a
cat (1 - n files)
cd (~, ..)
cp
grep
    not working: -l
head (-n)
less
mkdir
mv
pwd
rm
    not working: -r
rmdir
sort
tail (-n)
wc (-l, -m, -w)
who
history
!x
chmod
```
